## Usage

```bash
# Development
$ npm run start
$ npm run test

# Staging
$ npm run build
$ npm run preview
$ npm run deploy:staging

# Production
$ npm run version

# Render trames-noires drawing to video
$ npm run render -- <UID>
$ npm run render -- --help
```
